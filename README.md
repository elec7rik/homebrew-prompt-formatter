# homebrew-prompt-formatter

Homebrew tap for [prompt-formatter](https://github.com/elec7rik/prompt-fmt).

## Installation

```bash
brew tap elec7rik/prompt-formatter
brew install prompt-formatter
```

## Usage

```bash
prompt-formatter "fix the login bug"
```

See the [main repository](https://github.com/elec7rik/prompt-fmt) for full documentation.
